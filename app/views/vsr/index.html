Service Readiness prototype

This prototype journey is for the Service Readniess team to view the Royal Blue service in HAS.

Release: 3.2.0
Date: 27 Nov 2024

Visible to users

  • a. View appointments on one screen for both Assessment Centres
  • b. Improve date search for an appointment on the booked appointment page
  • c. Return to claimant details link – ability to move to the claimant case directly from the booked appointment list
  • d. Ability to filter by HCP on the booked appointment page
  • e. Increased pagination size to allow more viewable booked appointments on one screen

Not visible to users

  • A large amount of technical work was released to cover security, accessibility, performance, vulnerabilities, component version checks, compliance issues, automated testing improvements and a bug fix.
Some features like filters, logic and validation won't work the same way they will in the live service. This is a prototype. If you see an error page, it may be because you did not enter any information on the previous page.

Use Amy Gill (HCP) and Jane Doe (claimant) to view individual examples of managing the HCP profile and booking appointments.
